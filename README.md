A highly interactive, cyberpunk-inspired personal portfolio designed for **Ali Abdo**, a Remote Freelance Front-End Architect & Data Engineer. Built natively with Next.js App Router, Framer Motion, and Tailwind CSS.

This portfolio isn't just a document; it's a showcase of advanced front-end engineering, featuring custom graphics pipelines, real-time interactivity, and optimized UX patterns.

---

## ✨ Standout Features

- **🌐 Advanced Canvas Cursor:** A custom HTML5 Canvas cursor that interpolates colors dynamically on scroll (from cyan to violet) and casts an ambient spotlight onto the DOM.
- **⌨️ CMDK Command Palette:** Hit `Cmd + K` (or `Ctrl + K`) anywhere to open a global search bar and instantly teleport across the app, toggle modes, or execute commands.
- **🖥️ Interactive Terminal Emulator:** A fully functional terminal interface directly on the timeline that accepts bash-like commands (`help`, `about`, `skills`, `clear`, `hire`).
- **🎮 Konami Code Easter Egg:** Press `↑ ↑ ↓ ↓ ← → ← → B A` to trigger a fullscreen Matrix digital rain takeover effect.
- **🌍 CSS 3D Animated Globe:** A custom-built, CSS-only spinning globe visualization to represent global asynchronous workflow capacity.
- **📬 Serverless Contact Form:** Fully integrated with `EmailJS` with built-in validation, loading states, and custom glowing neon alerts.
- **📱 Responsive & Accessible:** Thoughtfully adapted for mobile architectures, utilizing native bento-grid layouts and touch-friendly target sizing.

## 🚀 Getting Started Locally

To get a local development environment up and running, follow these steps:

### 1. Clone the repository

```bash
git clone https://github.com/Aliabdo6/portfolio.git
cd portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🧭 Navigation & Shortcuts

This application supports heavy keyboard navigation for power users:

- `Cmd + K` / `Ctrl + K`: Open Command Palette
- `T`: Toggle Dark / Light Theme
- `?`: Open Shortcuts Help Menu
- `G` then `H`: Go Home
- `G` then `P`: Go to Projects
- `R`: Trigger Easter Egg (Matrix Mode)
